# Preenchimento do Figma
Nesta etapa, vamos iniciar o preenchimento do Figma com base nas atividades semanais desenvolvidas pelo grupo. O objetivo é organizar visualmente as ideias, protótipos e fluxos construídos ao longo das semanas, garantindo maior clareza e alinhamento entre todos os integrantes.

Esse processo permitirá consolidar o progresso do projeto, facilitando a comunicação, a validação das soluções propostas e a evolução contínua do design.

Nível de dificuldade: Médio
Link do Template MDS Squad 1: https://www.figma.com/board/13SnyvGleeaYsubnOhMeYk/Template-MDS-Squad-1?node-id=0-1&t=msR7ImIb1LnXBApl-1

## Semana 1

Na primeira semana, realizamos a formação dos grupos para o projeto de desenvolvimento de software, estabelecendo as equipes de trabalho e promovendo a integração entre os integrantes.

Além disso, iniciamos a organização das atividades e dos estudos em conjunto com a professora orientadora, alinhando expectativas, definindo os primeiros direcionamentos do projeto e estruturando o planejamento inicial para as próximas etapas.

Esse momento foi fundamental para criar uma base sólida de colaboração e entendimento sobre os objetivos do projeto.
