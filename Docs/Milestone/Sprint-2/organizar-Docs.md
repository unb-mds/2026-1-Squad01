## 📋 Descrição
Esta issue tem como objetivo estruturar e padronizar a documentação do projeto, com foco na criação e melhoria do README principal e na organização dos arquivos dentro da pasta de documentação.

A documentação deve apresentar de forma clara o propósito do projeto, suas funcionalidades, instruções de uso e organização interna, facilitando o entendimento tanto para novos colaboradores quanto para usuários externos.

Além disso, será definida uma estrutura padrão para os documentos, garantindo consistência e facilidade de manutenção ao longo do desenvolvimento.

## ✅ Tarefas
Criar ou atualizar o README principal do projeto
Adicionar descrição clara do projeto e seus objetivos
Organizar a pasta Docs
Revisar e atualizar documentações já existentes
